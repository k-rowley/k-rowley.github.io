Welcome to Keith's GitHub Home


Thursday, Feb 8, 2024

I just got started here at GitHub mainly because I teach HTML/CSS and am looking for a web hosting platform to publish student work. Over years and years of teaching computers, I've learned that file management is one of the most challenging skills and concepts for computer users to master. And managing web files is critical to successful web development. Over the past year, I've gone from using free web hosts such as FastComet and AwardSpace to using the online IDE Replit. But it appears that Replit has now made their hosting a paid service.

Then last week, I had a student submit their work with a GitHub URL, and that's when I realized that GitHub is not just for hosting application development projects, but also HTML/CSS projects.

I like the GitHub Desktop app, which helps web developers with their file management. GHD is kind of like an FTP program, which uploads one's local code to one's GitHub repository, but it's considerably easier to use than an FTP program, because it syncs local file changes with the repository, and one doesn't have to make sure to *manually* upload locally-edited files -- the Desktop app keeps track of locally-changed files and then uploads those upon command.
